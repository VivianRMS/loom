# Your Custom Skills

This folder contains your personalized, vault-specific versions of Loom skills.

## How Skills Get Here

When you run `LOOM!!!` in Claude and customize a skill, Claude creates a version here. For example:

- `video-research.md` — Your customized video research skill (folder paths, summary sections, etc. tailored to your vault)

## Using Your Custom Skills

Use the files in this folder, not the generic ones in `general-skills/`. Your custom versions have your setup baked in and are ready to use immediately.

## Example

You customized Video Research with:
- Research folder: `learning/videos`
- Summary sections: Summary, Investment Thesis, Risk Assessment

Your `video-research.md` here will have those values already filled in. Open it and follow the instructions to start using it.

---

**Don't edit these manually.** If you want to customize a skill differently, just run `LOOM!!!` again with Claude and it'll create (or overwrite) the version here.
